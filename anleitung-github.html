Was die GitHub-Version unterscheidet

Inhaltlich ist es dasselbe Notenbuch — Klassen, Noten, Berechnung, Export funktionieren identisch. Der einzige Unterschied ist, wie du die App öffnest und speicherst:

  • Statt eine notenbuch.html per Doppelklick zu öffnen, rufst du eine Web-Adresse auf: gehess-collab.github.io/notenbuch/notenbuch.html.
  • Die Seite bei GitHub liefert nur die leere App-Hülle aus — also Layout und Rechenlogik, aber keine Inhalte. Deine Noten werden nie irgendwohin hochgeladen; sie existieren ausschließlich in der .json-Datei, die du selbst auf deinem Gerät anlegst.
  • Das Speicherverhalten hängt vom Browser ab (Schritt 2) — auf iPad/iPhone läuft es etwas anders als am PC, weil Safari die dafür nötige Browser-Funktion nicht unterstützt.
ℹ️ Wann diese Version statt der lokalen Datei? Auf einem PC mit Chrome oder Edge kannst du wählen: lokale notenbuch.html oder GitHub-Adresse — beide verhalten sich dort identisch. Auf iPad/iPhone ist die GitHub-Adresse die einzig zuverlässige Variante, da Safari bei lokal geöffneten HTML-Dateien JavaScript nicht sicher ausführt.
1

Starten & zum Startbildschirm hinzufügen

Rufe die Adresse oben im Browser auf. Beim ersten Start ist alles leer — das ist normal, du legst deine Daten gleich selbst an (Schritt 3).

ℹ️ Empfohlen: als App-Symbol anlegen Auf iPad/iPhone: Teilen-Symbol ⬆️ antippen → „Zum Home-Bildschirm". Danach startet das Notenbuch per Fingertipp im Vollbild, ohne Adressleiste, wie eine echte App. Am PC genügt ein Lesezeichen auf die Adresse.
2

Speichern verstehen (nach Browser unterschiedlich)

Wie beim lokalen Notenbuch liegen deine Noten nicht im Browser, sondern in einer eigenen .json-Datei. Wie diese Datei geschrieben wird, hängt aber davon ab, was dein Browser kann:

Chrome / Edge (PC, Android)

Funktioniert genau wie die lokale Version: Klicke einmal auf Speichern und wähle einen Speicherort für deine Datei, z. B. 9b_deutsch.json. Danach speichert der Browser jede Änderung automatisch in diese Datei (Status oben in der Kopfleiste). Beim nächsten Mal: Datei öffnen → Datei auswählen.

iPad / iPhone (Safari) & andere Browser ohne Dateisystem-Zugriff

Diese Browser dürfen nicht direkt in eine Datei schreiben. Speichern öffnet stattdessen das Teilen-Menü von iOS/iPadOS:

  1. Beim allerersten Mal: Klasse(n) anlegen, dann auf Speichern tippen. Im Teilen-Menü „In Dateien sichern" wählen und einen Ort aussuchen (z. B. Auf meinem iPad oder iCloud Drive → Ordner „Notenbuch"). So legst du deine .json-Datei einmalig an.
  2. Ab jetzt: Datei zu Beginn der Sitzung über Datei öffnen laden.
  3. Es gibt kein automatisches Speichern. Nach jeder Änderung — neue Note, neue Schülerin, geänderte Gewichtung — musst du selbst auf Speichern tippen und im Teilen-Menü wieder „In Dateien sichern" wählen. Wähle dort denselben Ordner und denselben Dateinamen wie beim letzten Mal — nur dann fragt iOS „Ersetzen?" und überschreibt die bestehende Datei, statt eine Kopie anzulegen.
⚠️ Bricht das Teilen-Menü ab? Wenn du das Teilen-Menü ohne Auswahl schließt, bleibt die Änderung ungespeichert (kein zusätzlicher Download). Einfach erneut auf Speichern tippen.
⚠️ iOS schlägt einen neuen Namen vor (z. B. „notenbuch 2")? Dann ändere den Namen im Sichern-Dialog vor dem Speichern von Hand zurück auf den Originalnamen (z. B. notenbuch.json) — sonst legt iOS eine zweite, unabhängige Datei an und deine Noten landen an zwei Orten.
⚠️ Sicherung nicht vergessen Die .json-Datei ist der einzige Ort, an dem deine Noten existieren. Sichere sie regelmäßig — auf iPad/iPhone z. B. per iCloud-Drive-Synchronisierung oder indem du sie dir selbst per Mail/AirDrop schickst; am PC reicht eine Kopie auf einen USB-Stick.
3

Eine Klasse anlegen

Klicke oben links auf + Klasse. Trage Bezeichnung (z. B. „9b"), Fach und Schuljahr ein und wähle das Notensystem:

  • Note 1–6 — für die Sekundarstufe I. Es gibt Halbjahresnoten (Viertelnoten) und eine Jahresnote.
  • Notenpunkte 0–15 — für die Kursstufe. Jedes Halbjahr ist eine eigenständige Endnote, es gibt keine Jahresnote.
⚠️ Wichtig Das Notensystem ist nach dem Anlegen fest — sonst würden bereits eingetragene Noten falsch interpretiert. Für einen Kurs mit Notenpunkten legst du einfach eine weitere Klasse an.
4

Schüler/innen eintragen

Wechsle oben auf den Reiter Schüler. Tippe oder füge die Namen in das große Textfeld ein — ein Name pro Zeile, z. B. „Müller, Anna". Ein Klick auf Hinzufügen übernimmt alle auf einmal; die Liste wird automatisch alphabetisch sortiert.

5

Eine Leistung anlegen

Unter Leistungen → + Neue Leistung legst du eine Klassenarbeit, einen Test oder eine mündliche Note an:

  • Art — die Kategorie (Klassenarbeit, Test, Mündlich, GFS …). Welche es gibt, stellst du unter „Gewichtung" ein (Schritt 8).
  • Faktor — gewichtet die Leistung innerhalb ihrer Kategorie. Ein Test mit Faktor 0,5 zählt halb so viel wie einer mit Faktor 1.
  • Erfassung — entweder Punkte + Teilaufgaben (die Note wird über einen Notenschlüssel berechnet) oder Note direkt (du trägst die fertige Note ein, z. B. für mündliche Noten).
6

Punkte eingeben & Notenschlüssel

Nach dem Anlegen öffnet sich die Eingabemaske (später erreichbar über Noten eingeben). Von oben nach unten:

  1. Teilaufgaben — lege für jede Aufgabe die maximalen Verrechnungspunkte (VP) fest. Die Gesamtpunktzahl ergibt sich automatisch.
  2. Notenschlüssel — erzeuge ihn per Knopfdruck: Linear verteilt die Noten gleichmäßig; Mit Knick lässt dich die Bestehensgrenze frei wählen (z. B. Note 4,0 ab 50 %). Jeder Schwellwert in der Tabelle bleibt danach von Hand anpassbar.
  3. Punkteeingabe — trage je Schüler/in die Punkte pro Teilaufgabe ein. Summe und Note erscheinen sofort, unten läuft der Klassendurchschnitt (Ø) live mit.
ℹ️ Leere Felder sind okay Wer fehlt, bekommt einfach keinen Eintrag. Die Leistung wird für diese Person schlicht nicht mitgerechnet — es gibt keine automatische 6.
⚠️ Denk ans Speichern Auf iPad/iPhone (siehe Schritt 2) läuft hier nichts automatisch mit — nach einer Eingaberunde einmal auf Speichern tippen, damit nichts verloren geht.
7

Die Übersicht lesen

Der Reiter Übersicht ist das Herzstück: alle Leistungen, Halbjahres- und Jahresnoten auf einen Blick, oben die Kennzahlen der Klasse, unten der Durchschnitt jeder Spalte.

  • Die gerundete Note steht groß, die kleine graue Zahl daneben ist der ungerundete Dezimalwert.
  • Orange umrandet = Grenzfall (Dezimalwert nahe x,5). Klicke auf die Zelle, um die Note pädagogisch festzulegen.
  • Ein Punkt neben der Note zeigt: Diese Note wurde manuell festgelegt. Zum Zurücksetzen die Zelle anklicken und das Feld leer lassen.
8

Gewichtung einstellen

Unter Gewichtung bestimmst du, wie sich die Gesamtnote zusammensetzt. Jede Kategorie hat entweder eine eigene Gewichtung in Prozent (die Summe sollte 100 % ergeben) oder sie „zählt zu" einer anderen Kategorie — z. B. fließen Tests wie eine zusätzliche Klassenarbeit in deren Topf.

⚠️ Die Faktor-Regel bei „Zählt zu …" Sollen mehrere Tests zusammen wie eine zusätzliche Klassenarbeit zählen, gib jedem Test den Faktor 1 ÷ Anzahl der geplanten Tests (bei 4 Tests also 0,25 — wird beim Anlegen automatisch vorgeschlagen). Beim Standard-Faktor 1 zöge jeder einzelne Test so viel Gewicht wie eine ganze Klassenarbeit.

Kategorien, in denen ein/e Schüler/in noch keine Note hat, werden für sie automatisch ignoriert — die übrigen Gewichte rechnen anteilig hoch. Deine Übersicht stimmt also auch mitten im Schuljahr.

9

Exportieren & Drucken

Oben in der Übersicht findest du zwei Knöpfe:

  • CSV exportieren — erzeugt eine Tabelle für Excel/LibreOffice (z. B. für Zeugniskonferenzen). Auf iPad/iPhone landet sie ebenfalls im Teilen-Menü zum Sichern in „Dateien".
  • Drucken / PDF — öffnet den Druckdialog; dort „Als PDF speichern" wählen. Erklärtexte und Knöpfe werden im Ausdruck automatisch ausgeblendet.

Noch Fragen offen? Viele Antworten stehen direkt in der App: Die grauen Info-Boxen und aufklappbaren „Hinweise" erklären jede Ansicht an Ort und Stelle. Und weil alles in deiner eigenen Datei liegt: Datei kopieren = Backup, Datei auf ein anderes Gerät übertragen (z. B. per iCloud, Mail oder USB-Stick) = umgezogen — unabhängig davon, über welche Adresse du das Notenbuch öffnest.
